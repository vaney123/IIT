2) Open IDLE

  1. Press the Windows key and type IDLE (look for “IDLE (Python x.y)”). If your are using your own computer, your tutor will guide you to install the app.
  2. You’ll see the Python Shell window. Try typing 2 + 2 and press Enter.
  3. Open the Editor: File → New File. Save as coffee_shop.py.
  4. Run: press F5 or choose Run → Run Module.
Tip: The Shell is for quick experiments; the Editor is for full programs.

3) Step 1 — Your first print

Type this into the Editor and run it.

print("Welcome to the Python Coffee Shop!")
Concepts: print() shows text on screen. Text (a string) goes in quotes.

4) Step 2 — Ask for a name

customer_name = input("What is your name? ")
print("Hello, " + customer_name + "! Let's order some coffee.")
Concepts: input() reads text; a variable stores it. The + joins strings together.

5) Step 3 — Prices (numbers)

price_coffee = 3.50
price_latte = 4.00

print("Coffee: $" + str(price_coffee))
print("Latte: $" + str(price_latte))
Concepts: floats are numbers with decimals. str() converts a number to text so you can print it.

8) Step 6 — A short list

menu_items = ["coffee", "latte", "espresso"]
 print("Our menu:", menu_items)
Concepts: A list keeps many values in order under one name.

9) Put it together — Complete starter program

Copy all of this into coffee_shop.py and run it with F5.

print("Welcome to the Python Coffee Shop!")
 
 customer_name = input("What is your name? ")
 print("Hello, " + customer_name + "! Let's order some coffee.")
 
 price_coffee = 3.50
 price_latte = 4.00
 
 print("Coffee: $" + str(price_coffee))
 print("Latte: $" + str(price_latte))
 
 choice = input("What would you like to order? (coffee/latte): ")
 
 if choice == "coffee":
     cost = price_coffee
 elif choice == "latte":
     cost = price_latte
 else:
     print("Sorry, we do not have that.")
     cost = 0
 
 quantity = int(input("How many cups would you like? "))
 
 total_cost = cost * quantity
 
 if quantity > 1:
     print("You get a discount of $1.00!")
     total_cost -= 1.00
 
 print("Your total is: $" + str(total_cost))
 print("Thank you, " + customer_name + "! Please come again.")
Try It: Change prices and run again. What happens if you type a drink that isn't on the menu?

11) Mini‑glossary

  • String: text like "hello".
  • Integer: a whole number like 3.
  • Float: a decimal number like 3.5.
  • Variable: a named box that stores a value.
  • List: an ordered collection like ["a", "b", "c"].
